ROS2 + Gazebo + PX4 Basic Setup (Pop!_OS 24.04 / Ubuntu 24.04 Noble)

SYSTEM

OS: Pop!_OS 24.04 LTS (Ubuntu 24.04 Noble base) ROS 2 Distribution:
Jazzy

====================================================================== 1.
ROS 2 JAZZY INSTALLATION
======================================================================

Update system

sudo apt update

Install ROS 2 Desktop

sudo apt install ros-jazzy-desktop

Source ROS

source /opt/ros/jazzy/setup.bash

Verify ROS installation

ros2 doctor –report

======================================================================
2. RViz2 (Wayland Fix)
======================================================================

If RViz crashes under Wayland:

QT_QPA_PLATFORM=xcb rviz2

======================================================================
3. Gazebo (gz-sim / Harmonic)
======================================================================

Install Gazebo Harmonic

sudo apt install gz-harmonic

Verify Gazebo

gz sim

======================================================================
4. ROS ↔ Gazebo Bridge
======================================================================

Install ROS-GZ packages

sudo apt install ros-jazzy-ros-gz sudo apt install
ros-jazzy-ros-gz-bridge sudo apt install ros-jazzy-ros-gz-sim sudo apt
install ros-jazzy-ros-gz-image sudo apt install
ros-jazzy-ros-gz-interfaces

Test launch

ros2 launch ros_gz_sim_demos diff_drive.launch.py

Verify topics

ros2 topic list

======================================================================
5. RTAB-Map (ROS 2)
======================================================================

sudo apt install ros-jazzy-rtabmap-ros

======================================================================
6. PX4-Autopilot Repository
======================================================================

cd ~ git clone https://github.com/PX4/PX4-Autopilot.git –recursive

cd ~/PX4-Autopilot git checkout 9ac03f03eb git submodule update –init
–recursive

======================================================================
7. PX4 System Dependencies
======================================================================

sudo apt install git cmake ninja-build build-essential ccache
python3-pip python3-jinja2 python3-empy python3-numpy python3-yaml
python3-dev python3-venv libxml2-dev libxslt1-dev libeigen3-dev
protobuf-compiler libprotobuf-dev libopencv-dev libgstreamer1.0-dev
libgstreamer-plugins-base1.0-dev python3-jsonschema python3-kconfiglib

======================================================================
8. PX4 Python Virtual Environment
======================================================================

cd ~/PX4-Autopilot python3 -m venv .venv_px4 source
.venv_px4/bin/activate

Install Python dependencies inside venv

pip install pyros-genmsg pip install kconfiglib

Verify installs

python -c “import genmsg; print(‘genmsg OK’)” python -c “import
kconfiglib; print(‘kconfiglib OK’)”

======================================================================
9. PX4 SITL Build Command
======================================================================

cd ~/PX4-Autopilot source .venv_px4/bin/activate make px4_sitl
